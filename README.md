# PROJECT Cook.Si.In-server
냉장고에 있는 식재료로 뭘해야 할지 답답하지 않으시나요?  
'쿡시인'을 이용해보세요!  
Cook.Si.In은 식재료를 검색하면 관련된 요리 레시피들이 나오는 플랫폼 입니다.

# Dependencies
Front-end
  - React Native
  - React Navigation
  - Expo
  - Axios

Back-end
  - nodejs
  - express
  - sequelize
  - mysql
  - Amazon EC2
  - AWS RDS
  - PM2
